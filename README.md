## 策略逻辑

### RSI 指标说明
```
RSI = 100 - (100 / (1 + RS))
RS = 平均涨幅 / 平均跌幅

RSI 范围: 0 ~ 100
- RSI > 70: 超买区 (价格可能下跌)
- RSI < 30: 超卖区 (价格可能上涨)
```

### Top 1 策略详解 (RSI_14_35_65_L20)
```
RSI周期: 14
超卖线: 35 (买入阈值)
超买线: 65 (卖出阈值)
杠杆: 20x
止损: 3%
止盈: 15%

开仓规则:
  - 做多: RSI < 35 → 价格超卖 → 买入开多
  - 做空: RSI > 65 → 价格超买 → 卖出开空

平仓规则:
  - 多头平仓: RSI > 65 → 超买 → 平多仓
  - 空头平仓: RSI < 35 → 超卖 → 平空仓
  - 止损: 亏损3%时强制平仓
  - 止盈: 盈利15%时强制平仓
```
